# Portfolio — Serigne Bamba Diop

## Description
Portfolio de développeur web réalisé dans le cadre du cours PHP & MySQL à l'ESTM.
Site multipage, responsive, conçu avec HTML, CSS et JavaScript.

## Structure du projet
```
portfolio/
├── index.html              ← Page d'accueil (hero, à propos, compétences, stats, parcours)
├── css/
│   └── style.css           ← Feuille de style principale
├── js/
│   └── main.js             ← JavaScript (animations, recherche, formulaires)
├── pages/
│   ├── projets.html        ← Page des projets avec recherche par mots-clés
│   └── contact.html        ← Page contact + formulaire demande de projet
├── images/                 ← Dossier pour les images (ajouter vos photos)
└── README.md
```

## Fonctionnalités
- Design responsive (mobile, tablette, desktop)
- Navbar fixe avec effet de transparence au scroll
- Menu hamburger sur mobile
- Animations d'apparition au scroll
- Compteur de statistiques animé
- Recherche de projets par mots-clés
- Formulaire de contact avec validation HTML
- Formulaire de demande de projet
- Design sombre et épuré

## Technologies utilisées
- HTML5
- CSS3 (Flexbox, Grid, animations, media queries)
- JavaScript (Vanilla)
- Google Fonts (Sora, DM Sans)
- Boxicons (icônes)
